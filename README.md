**Wplace快捷小工具简介**

总之能识别颜色然后快速填入，不是bot，只是个按键触发软件，但是很方便，能省很多时间，具体看下4和5功能就知道有多方便了

**核心功能模式**
（别问我为什么要取这种不明所以的中二名称，因为部分功能的描述很长啊，我根本不知道如何概括，就干脆这样了）

**1. 织色如缕**
就是特定情况下替换左键为空格，解决左右手不协调的人（比如我）的烦恼

**2. 摄色流转**
点击时会取色并立即填充
（快速触发i+左键+空格）；
长按时，滑动到相同颜色会自动填充
（对鼠标位置的像素取色，遇到重复的就触发空格）

**3. 绘彩巡游**  
点击时取色并立即填充；
无需长按，只需滑动，滑动到相同颜色会自动填充，还可以WASD键控制鼠标移动
（和上一个一样，不过换了个触发方式）

**4. 纵穿千影 (纵向填充模式) - 新增**  
按住Shift点击左键，则会先取当前颜色，再填充整列的对应颜色；
直接点击会使用对上一次取色颜色填充
（方法和上一个也类似，但是先取色整列，获取对应的坐标后，一次性移动鼠标过去挨个触发空格键）

**5. 横扫苍茫 (横向填充模式) - 新增**  
同上，不过是横向

**其他便捷操作**
使用鼠标中键可以拖动画布
